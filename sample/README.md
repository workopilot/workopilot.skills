# Sample Demo - Order Management & Todo System

This is a sample application demonstrating how to use Workopilot Skills to build a modern SaaS-style backend management system.

## Features

- **Order Management** - Full CRUD operations for orders
- **Todo List** - Task management with priorities and tags
- **Modern UI** - Glassmorphism design with smooth animations
- **RESTful API** - Flask backend with JSON storage

## Quick Start

### Prerequisites

- Python 3.8+
- Node.js 16+
- npm 8+

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Backend runs at: `http://localhost:5000`

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at: `http://localhost:3000`

### One-Click Start

**Windows:**
```cmd
start-backend.bat
start-frontend.bat
```

**Linux/macOS:**
```bash
./start-backend.sh
./start-frontend.sh
```

## Technology Stack

**Backend:**
- Flask 3.0
- Flask-CORS
- JSON storage

**Frontend:**
- Vue 3
- Vue Router
- Tailwind CSS
- Vite
